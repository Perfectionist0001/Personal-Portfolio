# Nikhil Singh — Personal Portfolio

> A fully responsive, single-page developer portfolio built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies.

🌐 **Live Site:** [https://perfectionist0001.github.io/Personal-Portfolio](https://perfectionist0001.github.io/Personal-Portfolio)

---

## Features

- **Dark / Light theme** with localStorage persistence
- **Animated hero** — warp-speed star lines, neural network canvas, floating avatar
- **Project showcase** — filterable cards with live screenshots and direct links
- **Skills section** — tech stack icons with animated tooltips
- **Certifications** — horizontal scroll carousel with lightbox viewer
- **Coding profiles** — LeetCode, Codolio, GitHub stats
- **Contact form** — mailto compose with toast notifications
- **Offline screen** — deep-space canvas animation with astronaut when network drops
- **Back-to-top** button, scroll progress bar, keyboard shortcuts (`/` to focus nav, `T` for theme, `K` for Konami easter egg)
- **Lazy loading** images via IntersectionObserver
- **3D card tilt** on desktop (disabled on touch devices)
- **Swipe support** for lightbox on mobile
- **Global error handler** for graceful failure

## Tech Stack

| Layer | Tech |
